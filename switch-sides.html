Switch Sides or Divs Position for Small Devices

If I want to shift the rightside div before leftside div on small devices, how would i do it via jquery?


Explanation: If the screen width or device width will be less than 1023px the rightside div will switch to left side and leftsidediv will switch to right. It has two conditions script check on document.ready (page completes loading) and takes the action if necessary and script checks if window is resized and takes action if necessary.

Achieved by using 

$(".right-side").insertBefore(".left-side");

This needs a check of if screen width is less than 991px (tablet) and also a second check on window resize. A better Window resize check can be found at third example of sliding div.
