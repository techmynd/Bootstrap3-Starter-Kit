rem instead of px
=================
Typography font sizing in Bootstrap 4 is done completely based on the rem unit of measurement.
With rem, all font sizes are relative to the root element (aka, the html tag or body tag).
So if you inspect element on a Bootstrap 4 page, you'll notice the html tag has font-size: 16px on it.
If you wanted your p tags to be that size, you could simply do:
p {
 font-size: 1rem;
}
Another example: If you want your h1 tags to be be 40px.
h1 {
    font-size: 2.5rem;  /* 16  2.5 = 40px */
    /* 16 comes from 16px from root */
}
