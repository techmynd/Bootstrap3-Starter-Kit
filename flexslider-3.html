Flex Slider - Multiple Sliders

Use above flexslider for sliding images. This example uses images. All images height should be same. The images height becomes the slider height. Flex slider is not good with text in slides unless its just one text over all slides.

Use class show-arrows with flexslider main div, to show permanent arrows.

Better NOT use it as 'carousel' or 'slider gallery' for product section. Its only good to show 'single image scroll' or if you want to use it as product gallery, do not use carousel with gallery because it needs fixed width of slide 'li' from script and its behavior (image dimension) is not proper as carousel.
