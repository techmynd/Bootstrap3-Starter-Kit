Owl Carousel - Multiple Owl Carousels

Use above owl carousel for logos etc. The logoes size should be same - atleast height. Otherwise we use a responsive container for logoes to restrict max height as we have done above.

The Problem / Solution

Problem with owl carousel is right left arrows and the dots. If you move dots up or down the position of arrows changes. And there is arrows wrapper div that has obstinate z-index that comes over the overlay text. The solution is to use custom next back arrows.
